5 Biggest Lessons:

INTRO SENTENCE OF LESSONS LEARNT

  1. Set the project up for success from the start. That means always using responsive and fluid design principles, establishing base grid systems early, and using system-design for colours, layout and typography.
  2. The most important product is the final deployed design, having a cool idea in a design tool loses its shine if it makes the product more difficult to use or not functional.
  3. Organise everything in a usable, clear way. This helps you keep working on and adding additional design functionality as you progress.
  4. With each different 'brand identity', each project has similar base foundations, especially in Typography, Layout and Spacing.
  5. Use a step by step process by starting with core functionality, then adding additional styles, layout and structure at each step. Think about the core goal of the product, build that function, then add on from there with required extensions.
